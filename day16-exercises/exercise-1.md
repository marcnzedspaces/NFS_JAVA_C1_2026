# Exercise 1

## Objective

Use AI to review code without immediately accepting the answer.

## Task

Choose **one backend file** from your project:

```
CourseService.java
CourseController.java
AuthService.java
UserService.java
```

Choose only **one method**.

## Prompt to Use

```text
Act as a senior Java Spring Boot code reviewer.

Review the following method.

Focus on:
1. readability
2. error handling
3. validation
4. testability

Rules:
1. Do not add new features.
2. Do not change the business logic.
3. Do not introduce new dependencies.
4. Explain the issues first.
5. Suggest only the top 3 improvements.

Code:
[paste your method here]
```

## Participant Output

Create notes using this format:

```markdown
## Exercise 1: AI Code Review

File reviewed:

Method reviewed:

AI suggestion 1:

AI suggestion 2:

AI suggestion 3:

Suggestion I agree with:

Suggestion I disagree with:

Reason:
```

## Instructor Note

Participants should not blindly copy the AI answer. They must explain which suggestion is useful and which one may be risky.
